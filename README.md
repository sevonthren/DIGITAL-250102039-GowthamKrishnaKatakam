# DIGITAL-250102039-GowthamKrishnaKatakam
A few DEEDS files and Verilog scripts as per the problems provided by the Electronics Club, IIT Guwahati for the Digital Module of Electrothon'26.

'electrothonp1.pbs' and 'electrothon2.pbs' are the DEEDS Digital Circuit files for CHALLENGE-1 PART A (4-bit ALU core) and CHALLENGE-1 PART B (4-bit synchronous Up-Down Counter) respectively. 

"electrothonp3.v" and "electronthon_p4.v" are the Verilog scripts for CHALLENGE-2 PART A (FSM-101) and CHALLENGE-2 PART B (Frame Packet Reciever) respectively.

"Electrothon'26-Digital-FSM.pdf" contains the state diagrams both the Verilog scripts.
